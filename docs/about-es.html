03 : Sobre mi

Historia Educativa

¡Hola! Soy un desarrollador web y un entusiasta de la tecnología. Me encanta crear cosas nuevas y desafiarme a mí mismo para aprender más cada día. Actualmente estoy estudiando Ingeniería de Software, pero también completé un curso completo en el "Sistema Educativo Nacional, Dirección General de Centros de Formación para el Trabajo", el cual me otorgó 5 certificados:

  • Diseño y desarrollo web: Donde utilicé el software de diseño Figma para crear proyectos web.
  • HTML y CSS
  • WordPress
  • Desarrollo de JavaScript
  • Aplicaciones Moviles (Android and iOS)

Desde entonces, he estado trabajando en varios proyectos de desarrollo web.

Mi especialidad es el desarrollo de aplicaciones web utilizando tecnologías como HTML, CSS, JavaScript, Python, y frameworks como Flask y Django. También tengo experiencia utilizando bases de datos MySQL para pequeños proyectos personales, utilizando Python como mi lenguaje de desarrollo preferido.

En mi tiempo libre...

Además de mi trabajo como desarrollador web, realmente disfruto de la fotografía, el levantamiento de pesas, la música y jugar videojuegos. Me gusta viajar y experimentar nuevas culturas, y creo que la diversidad es un valor fundamental para la sociedad.

Si tiene alguna pregunta o desea saber más sobre mí o mi trabajo, no dude en ponerse en contacto conmigo.
